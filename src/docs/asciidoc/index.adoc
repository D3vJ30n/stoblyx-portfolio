== 개요

스토블릭스(Stoblyx)는 책 속 문장을 디지털 기념비(Obelisk)처럼 기억하고, AI를 통해 숏폼 영상으로 재탄생시키는 플랫폼입니다.

=== 기술 스택
* Spring Boot 3.3.9
* Java 17
* MySQL 8.0.41
* Redis 7.0.15
* Spring Data JPA
* Spring Security + JWT
